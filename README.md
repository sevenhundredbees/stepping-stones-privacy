# Privacy Policy for Stepping Stones

Last updated: 12/23/2025

## Data Collection
Stepping Stones does not collect, store, or transmit any personal data to external servers. All information you enter (goals, prayers, notes) is stored locally on your device.

## Data Storage
- All user data is stored locally using Hive database
- No data is shared with third parties
- No analytics or tracking is performed

## Notifications
The app requests notification permissions to send reminders for scheduled goals. These notifications are generated locally and do not transmit data.

## Future Features
If cloud backup is added in a future update, users will be able to opt-in. Data will be stored securely using Firebase and only the user will have access to their data.
